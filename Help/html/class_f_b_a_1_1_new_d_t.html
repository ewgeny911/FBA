Detailed Description

Класс для сериализации/десериализации DataTable.

Этот класс нужен для того, чтобы сохранить строку DataTable в файл. Используется в функциях ExportDataTabletoFile и ImportDataTableFromFile. Лучше сериализовать свой класс, а не чистый Datatable так как в класс можно добавить свои поля, например TableName.
