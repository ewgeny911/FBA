Подробное описание

Это класс-потомок для всех форм - и главной MDI Parent и всех дочерних.

Часть функционала здесь для родительской формы, а часть для дочерних. Часть функционала должна работать когда форма является дочерней формой MDI, Часть - когда форма является формой SDI. Часть - когда форма является главной формой MDIContainer. При этом функционал общий и для родительской формы. Все сделано в одном классе.
